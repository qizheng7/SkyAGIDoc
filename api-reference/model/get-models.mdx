---
title: "Get all available models"
api: "PUT https://api.mintlify.com/api/get-models"
description: "This endpoint returns the full list of available models"
---

### Body

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<ResponseField name="models" type="object">
  This is the list of the model names
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request PUT 'https://api.mintlify.com/api/get-models' \
--header 'Authorization: Token <token>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": "1",
  "models": [
    "gpt-3.5-turbo",
    "gpt-4",
    "llama-13b",
    ......
  ]
}
```

</ResponseExample>
