---
title: "Send a chat message"
api: "PUT https://api.mintlify.com/api/chat"
description: "This endpoint sends a chat to the agent."
---

### Body

<ParamField body="chat_id" type="string">
  This is the ID of the chat.
</ParamField>

<ParamField body="message" type="string">
  This is the chat message sent to the agent.
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<ResponseField name="resp_msg" type="string">
  This is the response from the agent.
</ResponseField>

<ResponseField name="agent_new_memory" type="object">
  The new memory of the initiate agent
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request PUT 'https://api.mintlify.com/api/chat' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "chat_id": "xxxxxxxxxxxx",
    "agent_message": "hello Leonard."
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": 1,
  "resp_msg": "hello. How are you?",
  "agent_new_memory": ["hello. How are you?"]
}
```

</ResponseExample>
