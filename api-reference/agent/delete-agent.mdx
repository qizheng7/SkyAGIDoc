---
title: "Delete an agent"
api: "PUT https://api.mintlify.com/api/delete-agent"
description: "This endpoint deletes an existing agent."
---

### Body

<ParamField body="agent_id" type="string">
  This is the ID of the to-delete agent.
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request PUT 'https://api.mintlify.com/api/delete-agent' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
		"agent_id": "xxxxxxxxxxxxxxx"
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": 1
}
```

</ResponseExample>
