---
title: "Send conversation message"
api: "PUT https://api.mintlify.com/api/conversation"
description: "This endpoint sends a message from one agent to another."
---

### Body

<ParamField body="conversation_id" type="string">
  This is the ID of the conversation.
</ParamField>

<ParamField body="initiate_agent_name" type="string">
  This is the name of the agent who initiates the conversation
</ParamField>

<ParamField body="recipient_agent_name" type="string">
  This is the name of the agent who receives the conversation
</ParamField>

<ParamField body="agent_message" type="string" optional>
  This is the message sent from the initiate agent to the recipient agent. This
  is mutual exclusive to the system_message.
</ParamField>

<ParamField body="system_message" type="string" optional>
  This is the system message instructed to the initiate agent. This is mutual
  exclusive to the agent_message.
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<ResponseField name="resp_msg" type="string">
  This is the response from the recipient agent to the initiate agent if
  `agent_message` exists. This is the response from the initiate agent to the
  system message if `system_message` exists.
</ResponseField>

<ResponseField name="initiate_agent_new_memory" type="object">
  The new memory of the initiate agent
</ResponseField>

<ResponseField name="recipient_agent_new_memory" type="object">
  The new memory of the recipient agent
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request PUT 'https://api.mintlify.com/api/conversation' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "conversation_id": "xxxxxxxxxxxx",
    "initiate_agent_name": "Penny",
    "recipient_agent_name": "Leonard",
    "agent_message": "hello Leonard."
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": 1,
  "resp_msg": "hello Penny. How are you?",
  "initiate_agent_new_memory": ["hello Leonard."],
  "recipient_agent_new_memory": [
    "hello Penny. How are you?",
    "I think Penny may like me."
  ]
}
```

</ResponseExample>
